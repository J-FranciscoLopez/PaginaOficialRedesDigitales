Práctica 4: Introducción a la pagina WEB

Juan Carlos Guauxochitl, y Jose Francisco López
Estudiantes de Ingeniería Mecatrónica
Fecha: Primavera 2024
Protocolos Arduino

- Introducción -

En el ámbito del desarrollo web, entender cómo funciona la creación de una página web es fundamental. En esta práctica, se enfocará en utilizar GitHub para construir una página web simple con un formato mínimo. El objetivo principal es comprender los conceptos básicos de desarrollo web y aprender cómo armar una página utilizando herramientas como GitHub. La página web resultante será muy sencilla, con un solo título, unos pocos subtitulos, un breve texto, una imagen y un enlace a la página de búsqueda de Google. Además, se explorará la posibilidad de acceder a una segunda página con un formato similar, pero con contenido diferente.

GitHub

- Objetivo General -

El objetivo general de esta práctica es comprender y experimentar con el proceso de creación de una página web utilizando GitHub. Se busca obtener una comprensión básica de los conceptos fundamentales del desarrollo web y familiarizarse con las herramientas y tecnologías involucradas en la creación de sitios web simples.

- Objetivos Especificos -

Crear una página web simple utilizando HTML y CSS.

Aprender a utilizar GitHub para alojar y gestionar proyectos de desarrollo web.

Experimentar con la estructura y el diseño de una página web básica.

- Desarrollo -

Para llevar a cabo esta práctica, se utilizará GitHub, una plataforma de desarrollo colaborativo que permite gestionar proyectos de software y alojar el código de manera gratuita. El proceso se dividirá en los siguientes pasos:
1. Configuración de GitHub: Se creará un nuevo repositorio en GitHub para el proyecto de la página web. Esto proporcionará un espacio para alojar los archivos HTML, CSS y otros recursos necesarios.
2. Creación de archivos HTML y CSS: Se escribirán los archivos HTML y CSS necesarios para construir la estructura básica de la página web. Esto incluirá el título, subtitulos, texto breve, imagen y enlace a la página de búsqueda de Google. El diseño será simple y minimalista, centrándose en comprender los conceptos básicos de la creación de páginas web.
3. Implementación de la funcionalidad de navegación: Se agregará un enlace en el breve texto que permita acceder a una segunda página con un contenido diferente, pero con el mismo formato que la original. Esto ayudará a explorar cómo funciona la navegación entre páginas web.
4. Prueba y ajustes: Se realizarán pruebas para asegurarse de que la página web se vea correctamente en diferentes dispositivos y navegadores. Se realizarán ajustes según sea necesario para mejorar la apariencia y la funcionalidad.
