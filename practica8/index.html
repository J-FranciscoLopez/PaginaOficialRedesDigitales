- Introducción -

En esta práctica se aplican conocimientos y elementos necesarios para configurar una red local eficiente y facilitar el acceso a recursos compartidos entre equipos interconectados. Este proceso se llevará a cabo utilizando el programa Packet Tracer de Cisco, una herramienta que permite simular redes y dispositivos de red.
Los objetivos específicos se centran en la creación de una red de computadoras para una escuela. En este escenario, se pretende simular una red aplicando direcciones IP de la subred 192.168.8.0 a las computadoras distribuidas en diferentes salas, tanto para aulas de cómputo como para la sala de administración. La verificación de la conectividad entre las computadoras se llevará a cabo mediante la utilización de la función ping desde la ventana de comandos.
Además, se presenta un desafío específico relacionado con la infraestructura de red en la escuela. Se describen las necesidades de la escuela, que incluyen 4 salas de cómputo con 12 computadoras cada una y una sala de administración con 7 computadoras para el personal administrativo. Se destaca la distancia física de más de 100 metros entre las salas de cómputo y la oficina administrativa. Para superar este desafío, se propone la implementación de repetidores y switches en la red para garantizar una conectividad eficiente y confiable. Además, se menciona la presencia de un router en la oficina administrativa que facilitará la conexión a Internet para todos los equipos de la escuela.

- Objetivo General -

Aplicar los elementos que se utilizan para configurar una red local y el acceso a recursos de un equipo en red.

- Objetivos Especificos -

• Utilizando el programa de Packet Tracer de Cisco, simula una red de aplicación, asignando a las computadoras las IP pertenecientes a la red 192.168.8.0., utilizando la ventana de comandos utiliza la función ping para verificar la comunicación entre cada una de las computadoras.

En una escuela se necesitan desarrollar una red de computadoras, se cuenta con 4 salas de cómputo con 12 computadoras cada uno, y una sala de administración con 7 computadoras para el personal administrativo. Los salones de cómputo se encuentran a más 100 metros de la oficina administrativa. La oficina administrativa posee un router para la conexión a internet de todos los equipos de la escuela. Utiliza repetidores y switches necesarios para desarrollar la red.
